# AI Engineer World's Fair Recommended Schedule

Personal four-day schedule for AI Engineer World's Fair 2026, focused on Codex workflows, quality gates, AI-native productivity, infrastructure/security, forward deployed engineering, and context engineering.

## Files

- `index.html` - GitHub Pages entry point.
- `outputs/ai-engineer-recommended-schedule.html` - Full standalone HTML schedule with day tabs and session-detail links.

## Publish To GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set the source to the branch you pushed, usually `main`, and root folder.
4. Open the published Pages URL.

The page is static HTML/CSS and does not require a build step.
